# PlantPal 🌱

**Удобный трекер ухода за комнатными растениями**

### Возможности

- Добавление и просмотр своих растений
- Подробная карточка каждого растения
- Журнал ухода (полив, удобрение, пересадка, опрыскивание, обрезка и т.д.)
- Выбор из 8 категорий растений
- Простая и понятная карточка на главном экране с последними записями ухода

### Стек технологий

- **Frontend**: Flutter (Web + Android)
- **Backend**: FastAPI + SQLAlchemy + PostgreSQL
- **Авторизация**: JWT
- **State management**: Provider
- **Графики и UI**: Material Design + fl_chart (в будущем)
- **Деплой**: Docker + docker-compose

### Как запустить локально

```bash
# 1. Клонировать репозиторий
git clone https://github.com/твой_юзер/plantpal.git
cd plantpal

# 2. Запустить все сервисы
docker-compose up --build
```

### Приложение будет доступно по адресу:

- Фронтенд: http://localhost:8080
- API (Swagger): http://localhost:8000/docs

### Основные функции

- Регистрация / вход
- Добавление растений с категорией
- Добавление записей ухода (полив, удобрение и др.)
- Просмотр истории ухода для каждого растения
